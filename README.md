Tuition Management System - UI

This repository contains the User Interface (UI) for a Tuition Management System. The UI is built with simplicity and compatibility in mind, using traditional web technologies such as Bootstrap, jQuery, and AJAX. It is designed to later integrate with a Spring Boot backend.

Features

Built using Bootstrap for responsive and modern UI components.

Utilizes jQuery and AJAX for dynamic content updates without page reloads.

Based on AdminLTE 3.2.0, providing a clean and professional dashboard layout.

Compatible with all integrated plugins, ensuring smooth functionality.

Simple architecture aimed at ease of understanding and maintenance.

Tech Stack

Frontend: HTML, CSS, JavaScript

Frameworks/Libraries: Bootstrap, jQuery, AdminLTE 3.2.0

AJAX: For asynchronous calls to backend APIs

Planned Backend Integration: Spring Boot
